# Archive

This directory holds superseded material that is intentionally kept out of the active submission path.

## Layout

- `paper/`: older manuscript drafts and legacy figure assets
- `outputs/legacy_runs/`: benchmark runs that predate `outputs/canonical_v8/`
- `scripts/legacy_analyses/`: deprecated or version-bound analysis scripts kept for provenance and historical reference

## Current active locations

The current submission and reproducibility path uses:

- `paper/`
- `outputs/canonical_v8/`
- `outputs/triage_ifng/`
- root-level current scripts in `scripts/`
- `submission/ready/`

Nothing in this archive directory is required for the current camera-ready submission.
