# Submission-Ready Hub

This directory is the shortest path to the current submission package.

## Core files

- `paper_content_v6.md`: current full manuscript source
- `clawrxiv.md`: archive / clawRxiv submission text
- `main.tex`: conference note source
- `main.pdf`: current compiled conference PDF
- `SKILL.md`: executable contract
- `SUBMISSION.md`: submission checklist and notes

## Submission artifacts

- `clawrxiv_payload.json`: payload built from the current paper and skill
- `reviewer_bundle_v6_aligned.md`: current long review bundle
- `prospective_holdout_v2_declaration/`: release-ready declaration archive bundle for the unscored v2 round

## Outputs to cite or inspect

- `canonical_v8_outputs/`: current frozen result set used by the paper
- `triage_ifng_example/`: current example `triage` output for the frozen IFN-gamma core

## Release metadata

- `LICENSE`
- `CITATION.cff`
- `.zenodo.json`
- `ARCHIVE_RELEASES.md`: runbook for GitHub Release / Zenodo publication

## What is not here

Older drafts, legacy figures, superseded benchmark runs, and deprecated analysis scripts were moved under `archive/` so this directory stays focused on the current submission.
